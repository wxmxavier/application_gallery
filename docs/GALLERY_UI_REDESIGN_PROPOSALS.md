# Application Gallery UI Redesign Proposals

**Date:** 2026-02-01
**Prepared by:** PM, UI/UX Design Team, Architecture Team
**Subject:** Gallery Interface Redesign with RSIP Integration

---

## Executive Summary

This document presents **three distinct UI design proposals** for redesigning the RSIP Application Gallery. Each proposal addresses:

1. **Better Organization** - Hierarchical task & scenario structure
2. **Improved Visual Design** - Modern, intuitive interface
3. **User Management** - SSO integration with RSIP platform
4. **Platform Integration** - Deep linking between Gallery and RSIP Wizard

---

## Current State Analysis

### Pain Points Identified
| Issue | Impact | Priority |
|-------|--------|----------|
| Flat organization | Hard to discover by "what robots do" | Critical |
| No user accounts | Can't save preferences/bookmarks | High |
| Basic filtering only | Power users can't refine searches | Medium |
| No RSIP integration | Gallery isolated from main platform | Critical |
| Mobile UX issues | Modal doesn't work on small screens | High |

### Current Metrics
- **487 total items** (123 videos, 194 images, 170 articles)
- **3 categories** (Industrial, Service, Security)
- **~15 task types**, **~20 capability types**
- **8 scene types** (warehouse, hospital, factory, etc.)

---

# Proposal 1: "Task Hub" - Hierarchical Task-Centric Design

## 1.1 Design Philosophy
**"What can robots DO?"** - Organize content primarily by robot tasks, with scenarios as secondary navigation.

```typescript
// Masonry layout calculation
interface MasonryItem {
  id: string;
  aspectRatio: '16:9' | '1:1' | '4:3' | '9:16';
  size: 'small' | 'medium' | 'large';
}

// Assign sizes based on engagement
function calculateCardSize(item: GalleryItem): MasonryItem['size'] {
  if (item.featured) return 'large';
  if (item.view_count > 10000) return 'medium';
  return 'small';
```

```typescript
interface WorkspaceState {
  wizard: WizardState;

    isOpen: boolean;

    filters: GalleryFilters;
    recentlyViewed: string[]; // item IDs
  };

// Context-aware filtering
function getContextFilters(wizard: WizardState): GalleryFilters {
  return {
    scene_type: wizard.selectedSceneTypeId
      ? mapToSceneType(wizard.selectedSceneTypeId)
      : undefined,
    task_types: wizard.moments
      .flatMap(m => m.selectedTaskIds)
      .map(mapToTaskType),
    requirements: wizard.selectedCapabilityIds
      .map(mapToRequirement),
  };
}
```
